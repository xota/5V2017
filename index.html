Je moet een website publiceren op github (Github Pages) en daarbij laten zien dat je ofwel veel CSS hebt geoefend of wel veel Javascript hebt gedaan. Bij de boordeling let ik in het bijzonder op

  1. Eigenheid, heb je iets zelf gemaakt of iets van iemand anders gekopierd
  2. Correctheid: werkt het, werken de links, plaatjes, functies, …
  3. Commentaar in zowel HTML, CSS als JS files die uitlegt waarom er staat wat er staat
  4. Inzet, al heb ik daar erg beperkte midleen voor. Het is wel belangrijk, zonder inzet leer je niets.

Voor CSS verwacht ik dat de Bootstrap cursus doet van freecodecamp.org

Voor Javascript de het op gametutorial gebaseerde boekje met opdrachten …
